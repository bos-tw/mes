## Bug 修復記錄 (2026-02-03)

### 問題 1：訂單最低委託額度警示未顯示
**根本原因**：CSS 中缺少 `.text-warning` 和 `.row-warning` 樣式
**修復**：在 `styles.css` 新增警示樣式
```css
.text-warning { color: #ff9800 !important; }
.row-warning { background-color: rgba(255, 152, 0, 0.08) !important; }
```

### 問題 2：庫存項目沒有顯示
**根本原因**：API 返回 `data.data` 但 JS 期待 `data.items`
**修復**：修改 `js/inventory_items.js` 第 258 行
- 修改前：`if (data.items) { renderTable(data.items); }`
- 修改後：`if (data.data) { renderTable(data.data); }`

### 問題 3：出貨單選擇客戶後訂單不顯示
**根本原因**：API 返回 `order.customer.id`（嵌套格式）但 JS 使用 `order.customer_id`（扁平格式）
**修復**：修改 `js/shipping_orders.js` 第 451 行
- 修改前：`state.orders.filter(o => String(o.customer_id) === String(customerId))`
- 修改後：`state.orders.filter(o => String(o.customer?.id || o.customer_id) === String(customerId))`
