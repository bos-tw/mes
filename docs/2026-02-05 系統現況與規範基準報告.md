## 3) 結構一致性基準（必守）

### 3.1 配置檔必備規範
- modal 必須包含 `hiddenFields: ['id']`
- 禁止不安全的 `querySelector(...).value =` 直接賦值
- JS 必須有 `setFieldValue()` 防禦性函數
- 混合模式 `requiresHtmlModal: true` 必須有對應 HTML

### 3.2 UI 統一規範（摘要）
- 按鈕：.btn primary / outline / text（表格內只用 btn text）
- Modal 尺寸：xlarge / large / medium / small（禁止 modal-window-large）
- 表單欄位：inline-label + full-width
- 表格欄位：data-column 與欄位設定一致

（詳見 docs/2026-01-28 2026-01-28 css-style-guide.md 與 .github/copilot-instructions.md）
