## 三、READ 操作風格分析

### 3.1 列表查詢（index.php GET）- 統一✅

**所有模組均使用以下標準結構**:

```php
// 1. 分頁參數處理
$page = max(1, (int)($_GET['page'] ?? 1));
$perPage = (int)($_GET['perPage'] ?? 10);
$perPage = min($perPage, 100);

// 2. 篩選條件
$conditions = ['deleted_at IS NULL'];  // 或 ['1=1']
$params = [];

if (!empty($_GET['keyword'])) {
    $conditions[] = 'name LIKE :keyword';
    $params['keyword'] = '%' . $_GET['keyword'] . '%';
}

// 3. 計算總數
$where = implode(' AND ', $conditions);
$countStmt = $pdo->prepare("SELECT COUNT(*) FROM table WHERE $where");
$countStmt->execute($params);
$total = (int)$countStmt->fetchColumn();

// 4. 查詢資料
$offset = ($page - 1) * $perPage;
$sql = "SELECT * FROM table WHERE $where ORDER BY id DESC LIMIT :limit OFFSET :offset";
$stmt = $pdo->prepare($sql);
foreach ($params as $key => $val) {
    $stmt->bindValue($key, $val);
}
$stmt->bindValue(':limit', $perPage, PDO::PARAM_INT);
$stmt->bindValue(':offset', $offset, PDO::PARAM_INT);
$stmt->execute();

// 5. 回傳結果
jsonResponse([
    'success' => true,
    'data' => $stmt->fetchAll(PDO::FETCH_ASSOC),
    'pagination' => [
        'page' => $page,
        'perPage' => $perPage,
        'total' => $total,
        'totalPages' => (int)ceil($total / $perPage),
    ],
]);
```

**✅ 優點**: 100% 統一，所有模組遵循相同模式

### 3.2 單筆查詢（show.php）- 風格差異 ⚠️

#### 模式 A：使用 `isset()` + 三元運算子（20 個模組）
```php
// quality_issue_reports/show.php
$id = isset($_GET['id']) ? (int)$_GET['id'] : 0;

if ($_SERVER['REQUEST_METHOD'] !== 'GET') {
    http_response_code(405);
    jsonResponse(['success' => false, 'message' => '僅支援 GET 請求。']);
}
```

#### 模式 B：直接使用 `??` 運算子（8 個模組）
```php
// customers/show.php (較新的檔案)
$id = (int)($_GET['id'] ?? 0);
```

#### 模式 C：先檢查 REQUEST_METHOD（12 個模組）
```php
// departments/show.php

    jsonResponse(['success' => false, 'message' => '僅支援 GET 請求。'], 405);
}

$id = (int)($_GET['id'] ?? 0);
```

**⚠️ 問題**: 參數處理順序不一致

### 8.1 立即改進項目（1-2 週內）

#### 1. 統一 DELETE 操作檢查方式
```php
// 建議統一格式（方案 A）
if ($_SERVER['REQUEST_METHOD'] !== 'DELETE') {
    http_response_code(405);
    jsonResponse(['success' => false, 'message' => '僅支援 DELETE 請求。']);

// 或（方案 B - 更彈性）
```

#### 2. 統一資料驗證方式
```php
// 建議統一使用 empty() 檢查必填欄位
function validateInput(array $data, bool $isUpdate = false): array {
    $errors = [];
    
    if (!$isUpdate || isset($data['name'])) {
        if (empty($data['name'])) {
            $errors[] = 'name 為必填欄位。';
        }
    }
    
    // 可選欄位使用 isset()

    }
    
    return $errors;
}
```
