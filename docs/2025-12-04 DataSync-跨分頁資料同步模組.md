# DataSync 跨分頁資料同步模組

## 概述

`DataSync` 是一個通用的跨分頁資料同步模組，用於處理 SPA（單頁應用）中不同 Tab 頁面之間的資料即時同步。當某個 Tab 頁面的資料發生變更（新增、修改、刪除）時，其他已開啟的相關 Tab 頁面會自動重新載入資料。

## 檔案位置

```
js/data-sync.js
```

## 同步機制

1. **同一瀏覽器視窗內的 Tab**：使用 `CustomEvent` 事件
2. **跨瀏覽器分頁**：使用 `localStorage` 事件（storage event）
3. **防抖處理**：避免頻繁重複更新（預設 300ms）
4. **依賴自動通知**：修改 A 表會自動通知使用 A 表資料的 B、C 表

## API 參考

### 事件類型常數

```javascript
DataSync.EVENT_TYPES = {
    CREATED: 'created',              // 資料新增
    UPDATED: 'updated',              // 資料更新
    DELETED: 'deleted',              // 資料刪除
    BULK_UPDATED: 'bulk_updated',    // 批量更新
    DEPENDENCY_UPDATED: 'dependency_updated'  // 依賴模組更新
};
```

### 核心方法

#### `DataSync.notify(module, action, data)`

發送資料變更通知。

| 參數 | 類型 | 說明 |
|------|------|------|
| module | string | 模組名稱，如 `'customers'`, `'orders'` |
| action | string | 操作類型，使用 `DataSync.EVENT_TYPES` |
| data | any | 變更的資料（可選） |

```javascript
DataSync.notify('customers', DataSync.EVENT_TYPES.UPDATED, { id: 1, name: '新名稱' });
```

#### `DataSync.notifyWithDependencies(module, action, data)`

發送通知並自動通知所有依賴該模組的其他模組。

```javascript
// 當客戶資料更新時，會自動通知 orders, screening_services 等依賴模組
DataSync.notifyWithDependencies('customers', DataSync.EVENT_TYPES.UPDATED, result.data);
```

#### `DataSync.subscribe(modules, callback)`

訂閱資料變更事件。

| 參數 | 類型 | 說明 |
|------|------|------|
| modules | string \| string[] | 要監聽的模組名稱，使用 `'*'` 監聽所有變更 |
| callback | Function | 回呼函數 `(action, data, module) => void` |

**返回值**：取消訂閱的函數

```javascript
// 訂閱單一模組
const unsubscribe = DataSync.subscribe('orders', (action, data, module) => {
    console.log(`${module} 發生 ${action} 事件`, data);
    loadOrders();
});

// 訂閱多個模組
DataSync.subscribe(['customers', 'suppliers'], (action, data, module) => {
    reloadDropdowns();
});

// 監聽所有變更
DataSync.subscribe('*', (action, data, module) => {
    console.log('有資料變更:', module, action);
});

// 取消訂閱
unsubscribe();
```

#### `DataSync.createModuleHelper(moduleName, options)`

建立模組輔助器，簡化整合流程（**推薦使用**）。

| 參數 | 類型 | 說明 |
|------|------|------|
| moduleName | string | 模組名稱 |
| options.onRefresh | Function | 資料重新載入函數 |
| options.onDependencyUpdate | Function | 依賴更新時的處理函數（可選） |
| options.debounceMs | number | 防抖延遲毫秒數，預設 300 |
| options.withDependencies | boolean | 發送通知時是否包含依賴，預設 true |

**返回值**：模組輔助器物件

```javascript
const helper = DataSync.createModuleHelper('customers', {
    onRefresh: () => loadCustomers(state.page),
    onDependencyUpdate: (sourceModule, sourceAction, sourceData) => {
        if (sourceModule === 'companies') {
            loadCompanyOptions();
        }
    },
    debounceMs: 300
});

// 輔助器提供的方法
helper.notifyCreated(data);    // 通知新增
helper.notifyUpdated(data);    // 通知更新
helper.notifyDeleted({ id });  // 通知刪除
helper.notifyBulkUpdated(data); // 通知批量更新
helper.refresh();              // 手動觸發重新載入
helper.destroy();              // 銷毀輔助器
```

## 模組依賴關係

當某個模組資料變更時，以下依賴模組會自動收到 `dependency_updated` 通知：

| 來源模組 | 依賴模組（會被通知） |
|----------|---------------------|
| `companies` | employees, customers, suppliers |
| `customers` | orders, screening_services |
| `suppliers` | orders, inventory_items |
| `screening_items` | order_items, screening_services, inventory_items |
| `screening_services` | orders, order_items |
| `employees` | work_orders, orders |
| `departments` | employees |
| `machines` | work_orders |
| `tools` | work_orders, order_items |
| `orders` | order_items, work_orders |
| `order_items` | work_orders, inventory_items |
| `work_orders` | work_order_images, work_order_first_piece_dimensions |
| `lookup_values` | orders, customers, suppliers, employees, work_orders, screening_items |

### 動態管理依賴

```javascript
// 註冊新的依賴關係
DataSync.registerDependency('new_module', ['dependent1', 'dependent2']);

// 新增依賴
DataSync.addDependency('customers', 'new_dependent_module');

// 查詢依賴
const dependents = DataSync.getDependentModules('customers');
// 返回: ['orders', 'screening_services']
```

## 整合指南

### 步驟 1：在 index.html 引入模組

確保 `data-sync.js` 在其他模組 JS 之前載入：

```html
<script src="script.js"></script>
<!-- 跨分頁資料同步模組 -->
<script src="js/data-sync.js"></script>
<!-- 其他模組 -->
<script src="js/customers.js"></script>
<script src="js/orders.js"></script>
```

### 步驟 2：在模組初始化時建立輔助器

在模組的初始化函數末尾加入：

```javascript
function initializeYourModule(container) {
    // ... 原有的初始化程式碼 ...

    loadData(1);

    // 建立資料同步輔助器
    if (typeof DataSync !== 'undefined') {
        DataSync.createModuleHelper('your_module', {
            onRefresh: () => loadData(state.page),
            debounceMs: 300
        });
    }
}
```

### 步驟 3：在 CRUD 成功後發送通知

#### 新增成功後

```javascript
const response = await fetch('api/your_module/index.php', { ... });
const result = await response.json();

if (result.success) {
    showAlert('success', '資料已新增。');
    loadData(1);

    // 發送資料同步通知
    if (typeof DataSync !== 'undefined') {
        DataSync.notifyWithDependencies('your_module', DataSync.EVENT_TYPES.CREATED, result.data);
    }
}
```

#### 更新成功後

```javascript
if (result.success) {
    showAlert('success', '資料已更新。');
    loadData(state.page);

    // 發送資料同步通知
    if (typeof DataSync !== 'undefined') {
        DataSync.notifyWithDependencies('your_module', DataSync.EVENT_TYPES.UPDATED, result.data);
    }
}
```

#### 刪除成功後

```javascript
if (result.success) {
    showAlert('success', '資料已刪除。');
    loadData(state.page);

    // 發送資料同步通知
    if (typeof DataSync !== 'undefined') {
        DataSync.notifyWithDependencies('your_module', DataSync.EVENT_TYPES.DELETED, { id });
    }
}
```

## 完整範例

### 範例：customers.js 整合

```javascript
(function() {
    'use strict';

    function initializeCustomersModule(container) {
        const state = { page: 1 };

        // 載入客戶資料
        async function loadCustomers(page) {
            // ... 載入邏輯 ...
        }

        // 儲存客戶資料
        async function saveCustomer(isEdit) {
            try {
                const response = await fetch(endpoint, { ... });
                const result = await response.json();

                if (result.success) {
                    showAlert('success', isEdit ? '客戶資料已更新。' : '客戶資料已新增。');
                    loadCustomers(isEdit ? state.page : 1);

                    // ✅ 發送資料同步通知
                    if (typeof DataSync !== 'undefined') {
                        DataSync.notifyWithDependencies(
                            'customers',
                            isEdit ? DataSync.EVENT_TYPES.UPDATED : DataSync.EVENT_TYPES.CREATED,
                            result.data
                        );
                    }
                }
            } catch (error) {
                console.error(error);
            }
        }

        // 刪除客戶
        async function deleteCustomer(id) {
            try {
                const response = await fetch(`api/customers/delete.php?id=${id}`, { method: 'DELETE' });
                const result = await response.json();

                if (result.success) {
                    showAlert('success', '客戶資料已刪除。');
                    loadCustomers(state.page);

                    // ✅ 發送資料同步通知
                    if (typeof DataSync !== 'undefined') {
                        DataSync.notifyWithDependencies('customers', DataSync.EVENT_TYPES.DELETED, { id });
                    }
                }
            } catch (error) {
                console.error(error);
            }
        }

        // 初始載入
        loadCustomers(1);

        // ✅ 建立資料同步輔助器
        if (typeof DataSync !== 'undefined') {
            DataSync.createModuleHelper('customers', {
                onRefresh: () => loadCustomers(state.page),
                debounceMs: 300
            });
        }
    }

    window.initializeCustomersModule = initializeCustomersModule;
})();
```

### 範例：處理依賴更新

```javascript
// orders.js - 需要在客戶資料變更時重新載入客戶下拉選單
if (typeof DataSync !== 'undefined') {
    DataSync.createModuleHelper('orders', {
        onRefresh: () => loadOrders(state.page),
        onDependencyUpdate: (sourceModule, sourceAction, sourceData) => {
            console.log('[orders] 依賴模組更新:', sourceModule);

            // 當客戶資料變更時，重新載入客戶下拉選單
            if (sourceModule === 'customers') {
                loadCustomerOptions();
            }

            // 重新載入訂單列表
            loadOrders(state.page);
        },
        debounceMs: 300
    });
}
```

```

[DataSync] 已訂閱: customers
[DataSync] 已發送通知: customers - updated {...}
[DataSync] 收到通知: customers - updated
[DataSync] 觸發 customers 資料重新載入
```

### 檢查當前 Tab ID

```javascript
console.log(DataSync.currentTabId);
```

### 取得模組輔助器

```javascript
const helper = DataSync.getModuleHelper('customers');
if (helper) {
    helper.refresh(); // 手動觸發重新載入
}
```

## 注意事項

1. **防抖機制**：預設 300ms，避免短時間內多次觸發重新載入
2. **自我排除**：Tab 發出的事件不會觸發自己的監聽器（使用 `sourceTabId` 識別）
3. **安全檢查**：所有呼叫都使用 `typeof DataSync !== 'undefined'` 檢查，確保相容性
4. **記憶體管理**：使用 `helper.destroy()` 可以取消訂閱並釋放資源
