安全設定

調整頁面更新偵測、自動登出及登入鎖定等安全相關功能

版本更新偵測

系統有新版本時，顯示橫幅提示使用者重新整理頁面以載入最新版本。

閒置自動登出

使用者長時間無操作時，顯示倒數警告並自動執行登出，保護帳號安全。

登入失敗鎖定

密碼錯誤次數過多時，暫時鎖定 IP 位址的登入請求，防止暴力破解。

系統更新

說明

  • 版本更新偵測：系統偵測到新版本時顯示橫幅，但不會強制重新整理，使用者自行決定是否更新。
  • 閒置自動登出：前端倒數計時器在到達逾時秒數前顯示警告，使用者可點擊「繼續使用」取消自動登出。儲存後立即生效，不需重新登入。
  • 登入失敗鎖定：鎖定依據 IP 位址判斷（非帳號），適用於防止網路暴力破解。關閉此功能請確認環境安全。
